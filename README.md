# Proyecto de Análisis de Salud Mental - NODO

## Descripción del Proyecto

Proyecto de análisis de datos sobre salud mental, desarrollado como parte del reto final de NODO. El proyecto incluye análisis exploratorio de datos, limpieza, almacenamiento en base de datos y visualización interactiva en Power BI.

## Dataset

El dataset utilizado es "Mental Illnesses Prevalence" que contiene información sobre la prevalencia de enfermedades mentales a nivel global. Los datos incluyen estadísticas sobre diferentes condiciones de salud mental en diversos países y periodos de tiempo.

Fuente de datos: `data/raw/1- mental-illnesses-prevalence.csv`
